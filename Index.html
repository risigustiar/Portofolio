  • Real-time color data display on mobile application
  • RGB color visualization for accurate comparison

  • User-friendly interface designed for tailors

Project 05
EduFarm
EduFarm is a digital agricultural education app that helps novice farmers and the general public learn farming through training materials, practical guides, live mentors, and supporting documentation. With EduFarm, users can find the best knowledge, tools, and guidance to improve their farming skills and grow better every day.
  • Displays user greetings, training recommendations, and live mentor schedules
  • Displays a list of farming training such as peanuts, corn, rice, wheat, and peas
  • Displays training information, material descriptions, and learning objectives
  • Users can read farming materials in a complete and focused manner
  • Displays the user's learning progress, such as the amount of material that has been studied
  • Displays live mentor schedule and live learning session information
  • Users can view or download supporting documents such as agricultural guides or journals
My Contribution

On the EduFarm project, I was involved in implementing the application design into the front-end interface and connecting the application to the database. I ensured that each main page, such as Home, Training, Material Details, Live Mentor, and Account, displayed well, was responsive, and aligned with the UI design. Furthermore, I integrated the application data so that information such as training lists, materials, progress, and user data could be managed through the database.
